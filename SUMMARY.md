# CalcLeap Embeddable Calculators - Build Summary

## ✅ Task Completed

Built a complete embeddable calculator widget system for CalcLeap.com to generate backlinks and provide value to other websites.

## 📦 What Was Created

### Main Showcase Page
**File**: `embed.html` (16KB)
- Professional landing page displaying all 5 calculators
- Live iframe previews of each widget
- One-click copy embed code snippets
- Apple-inspired design (#f5f5f7 bg, #0071e3 accent, -apple-system fonts)
- Mobile-responsive grid layout
- Feature highlights section
- CTA section linking back to CalcLeap

### 5 Functional Calculator Widgets

**1. BMI Calculator** - `embed/bmi.html` (6.5KB)
- ⚖️ Body Mass Index calculation
- Supports metric (kg/cm) and imperial (lbs/in)
- Color-coded categories: Underweight, Normal, Overweight, Obese
- Real-time unit switching

**2. Mortgage Calculator** - `embed/mortgage.html` (6.2KB)
- 🏠 Monthly mortgage payment calculation
- Shows total payment and total interest
- Proper amortization formula: M = P[r(1+r)^n]/[(1+r)^n-1]
- Number formatting with thousands separators

**3. Compound Interest Calculator** - `embed/compound.html` (7.3KB)
- 💰 Investment growth with compound interest
- Multiple compounding frequencies (annual, semi-annual, quarterly, monthly, daily)
- Optional monthly contribution support
- Shows future value, principal, and interest earned

**4. Tip Calculator** - `embed/tip.html` (8KB)
- 💵 Tip and bill splitting
- Quick-select buttons (10%, 15%, 18%, 20%)
- Custom tip percentage
- Bill splitting across multiple people
- Auto-calculation on input change

**5. Age Calculator** - `embed/age.html` (7.6KB)
- 🎂 Precise age from birthdate
- Shows years, months, days breakdown
- Total time in months, weeks, days
- Days until next birthday
- Custom calculation date support

```html

</div>
```

Standard embed code format:
```html
<iframe src="https://calcleap.com/embed/bmi.html" width="400" height="500" frameborder="0"></iframe>
<p style="text-align:center;font-size:12px;margin-top:8px;">Powered by <a href="https://calcleap.com" target="_blank">CalcLeap</a></p>
```
