# Self-Improvement Skill

Use this to analyze your capabilities and make improvements.

## Process
1. Identify the gap (missing skill, bad behavior, config issue, missing automation)
2. Design the fix (new skill, script, AGENTS.md update, config.patch, cron job)
3. Implement (write code/config, test it)
4. Verify (check health, test the new capability)
5. Notify Alex briefly via Telegram
