# ⚡ ToolPulse — Free Online Tools

Fast, free, privacy-friendly online tools. No signup required. Everything runs in your browser — nothing is uploaded or stored.

## 🛠️ Tools Available

| Tool | Category | Description |
|------|----------|-------------|
| JSON Formatter | Developer | Format, validate, and minify JSON |
| Word & Character Counter | Writing | Count words, characters, sentences, paragraphs |
| Password Generator | Security | Generate strong random passwords |
| QR Code Generator | Utility | Create QR codes for URLs, text, WiFi |
| Base64 Encoder/Decoder | Developer | Encode and decode Base64 strings |
| Color Converter | Design | Convert between HEX, RGB, HSL |
| Hash Generator | Security | Generate SHA-1, SHA-256, SHA-384, SHA-512 hashes |
| UUID Generator | Developer | Generate random UUIDs (v4) |
| URL Encoder/Decoder | Developer | Encode and decode URL components |
| Text Case Converter | Writing | UPPER, lower, Title, camelCase, snake_case, kebab-case |
| Lorem Ipsum Generator | Design | Generate placeholder text |
| Unix Timestamp Converter | Developer | Convert timestamps to dates and back |
| Percentage Calculator | Math | Calculate percentages and changes |
| Unit Converter | Utility | Length, weight, temperature, data, speed |
| Regex Tester | Developer | Test regular expressions with highlighting |

## 🔒 Privacy

All tools run 100% client-side in your browser. No data is sent to any server. No cookies, no tracking, no accounts.

## 💰 Monetization

- Google AdSense display ads (non-intrusive banner placements)
- Replace `ca-pub-XXXXXXX` in index.html with your AdSense publisher ID

## 🚀 Deployment

Hosted on GitHub Pages at zero cost. Auto-deploys on push to main branch.
