# Laser Family Office — North Star

## The one number that matters
**Hours Alex saves per week** by using the Laser dashboard for portfolio review, AUM tracking, and family-office reporting (target: 5+ hours/week).

This is not a revenue project — it's a **leverage** project. The dashboard exists so Alex can manage ~$920M AUM more efficiently in his CIO role.

## Current value
Unknown — needs measurement. The first job of the next strategic cycle on this project is to ask Alex: "How many hours did you save with the dashboard last week?"

## Threshold for "working"
Alex uses the dashboard at least 3 days/week without prompting and self-reports >3 hours/week saved.

## Threshold for "deprecate"
Alex hasn't opened the dashboard in 30 days. At that point, ask whether to invest more or shut it down.

## Hard rule
If usage telemetry shows 0 logins in 14 days, escalate to Alex: "Is Laser still useful? Should I keep maintaining it?"
